Class 11 HBSE Computer Science – Chapter 1: Computer Fundamentals

1. Introduction to Computer

A computer is an electronic device that can:

  • Accept input (data from user or device)
  • Process it according to instructions
  • Store the data for later use
  • Produce output in a useful form
Example:
Input: Marks of 5 subjects
Processing: Total & Average calculation
Output: Total = 450, Average = 90

Characteristics of Computer:

  • High Speed: Can perform millions or billions of calculations per second
  • Accuracy: Produces correct results if input and program are correct
  • Automation: Works automatically according to instructions
  • Versatility: Can perform multiple tasks like calculations, graphics, games, etc.
  • Storage & Retrieval: Can store huge amounts of data and retrieve it quickly
  • Connectivity: Can connect with other computers via networks

2. Evolution of Computers

Computers have evolved in 5 generations:

Generation Technology Examples Features
1st (1940-56) Vacuum Tubes ENIAC Huge size, slow, generated heat, consumed lots of electricity
2nd (1956-63) Transistors IBM 1401 Smaller, faster, reliable, less heat
3rd (1964-71) Integrated Circuits (IC) IBM 360 Compact, cheaper, reliable
4th (1971-Present) Microprocessors Personal Computers Very fast, portable, affordable
5th (Present & Future) AI, NLP AI systems Intelligent, self-learning, voice recognition

3. Computer Memory

Memory stores data and instructions:

  • Primary Memory (Fast & Volatile)
    • RAM: Temporary memory, volatile, example: programs currently running
    • ROM: Permanent memory, non-volatile, stores boot instructions
  • Secondary Memory: Stores data permanently, examples: HDD, SSD, Pen Drive, CD/DVD
  • Cache Memory: Very fast memory between CPU and RAM for quick access

4. Data Transfer between Memory and CPU

The CPU (Central Processing Unit) is the brain of the computer.

Buses for Data Transfer:

  • Data Bus: Transfers actual data
  • Address Bus: Tells CPU where to read/write
  • Control Bus: Sends control signals (Read/Write)

Data Flow Example:

Input → Memory → CPU → Processing → Output → Storage

5. Microprocessor

A microprocessor is a single-chip CPU that performs all operations of a computer.

  • ALU: Performs calculations and logical operations
  • CU: Directs the flow of data and instructions
  • Input/Output Operations: Communicates with devices

Uses: PCs, laptops, smartphones, calculators, washing machines, ATMs

Key Features: Fast, compact, versatile

6. Data and Information

  • Data: Raw facts, numbers, or letters. Example: 10, 20, 30
  • Information: Processed data that is meaningful. Example: Total marks = 150

Key Concept: Data → Processing → Information

7. Software

Software is a set of instructions that tells the computer what to do.

  • System Software: Helps the computer run
    • OS: Manages CPU, memory, storage, devices (Windows, Linux, macOS)
    • Utility Programs: Maintenance programs like Antivirus
  • Application Software: Helps users perform tasks (MS Word, Excel, PowerPoint)

Hardware vs Software:

HardwareSoftware
Physical componentsPrograms/instructions
Can be touchedCannot be touched
Example: CPU, MonitorExample: MS Word, Chrome

8. Operating System (OS)

The Operating System is a system software that acts as an interface between the user and hardware.

Functions of OS:

  • Resource Management: CPU, memory, storage
  • File Management: Organizes files & folders
  • Process Management: Runs multiple programs simultaneously
  • Device Management: Controls input/output devices
  • User Interface: Provides GUI for easy interaction

Examples: Windows, macOS, Linux, Android

9. Quick Summary / Revision Notes

  • Computer = Input + Processing + Output + Storage
  • Memory Types: RAM (temporary), ROM (permanent), Cache (fast), Secondary storage (HDD, SSD)
  • CPU/Microprocessor: Brain of the computer, does all processing
  • Data → Information: Data is raw; processing makes it meaningful
  • Software: System Software (OS, Utilities), Application Software (Word, Excel)
  • Operating System: Manages hardware, files, devices, processes, and provides GUI

Tip for Exam: Remember the flow: Input → Memory → CPU → Processing → Output → Storage → Information
