Class 11 HBSE Computer Science – Chapter 1

1. Introduction to Computer

A computer is an electronic device that can:

  • Accept input (data from user or device)
  • Process it according to instructions
  • Store the data for later use
  • Produce output in a useful form
Example: Input: Marks of 5 subjects; Processing: Total & Average calculation; Output: Total = 450, Average = 90

Characteristics of Computer:

  • High Speed: Millions or billions of operations per second
  • Accuracy: Produces correct results
  • Automation: Works automatically according to instructions
  • Versatility: Multiple tasks like calculations, games, graphics
  • Storage & Retrieval: Can store and retrieve huge amounts of data
  • Connectivity: Can connect to other computers via networks

2. Evolution of Computers

Computers evolved in 5 generations:

Generation Technology Examples Features
1st (1940-56) Vacuum Tubes ENIAC Large, slow, heat, high electricity consumption
2nd (1956-63) Transistors IBM 1401 Smaller, faster, reliable, less heat
3rd (1964-71) Integrated Circuits (IC) IBM 360 Compact, cheaper, reliable
4th (1971-Present) Microprocessors PCs Very fast, portable, affordable
5th (Present & Future) AI, NLP AI systems Intelligent, self-learning, voice recognition

3. Computer Memory

Memory stores data and instructions:

  • Primary Memory:
    • RAM: Temporary, volatile, example: open programs
    • ROM: Permanent, non-volatile, stores boot instructions
  • Secondary Memory: Permanent storage, examples: HDD, SSD, Pen Drive, CD/DVD
  • Cache Memory: Very fast memory between CPU and RAM for quick access

4. Data Transfer between Memory and CPU

CPU is the brain of the computer. It communicates with memory using buses:

  • Data Bus: Transfers actual data
  • Address Bus: Tells CPU where to read/write
  • Control Bus: Sends commands like Read/Write

Data Flow Example: Input → Memory → CPU → Processing → Output → Storage

5. Microprocessor

A microprocessor is a single-chip CPU that performs all operations:

  • ALU: Arithmetic & logic operations
  • CU: Directs flow of instructions
  • Input/Output Operations: Communicates with devices

Uses: PCs, laptops, smartphones, calculators, ATMs

Key Features: Fast, compact, versatile

6. Data and Information

  • Data: Raw facts, numbers, letters (Example: 10, 20, 30)
  • Information: Processed data, meaningful (Example: Total marks = 150)

Concept: Data → Processing → Information

7. Software

Software is a set of instructions for the computer:

  • System Software:
    • OS: Manages CPU, memory, storage, devices (Windows, Linux)
    • Utility Programs: Maintenance programs like Antivirus
  • Application Software: Helps users perform tasks (MS Word, Excel, PowerPoint)

Hardware vs Software:

HardwareSoftware
Physical componentsPrograms/instructions
Can be touchedCannot be touched
Example: CPU, MonitorExample: MS Word, Chrome

8. Operating System (OS)

  • Manages hardware, memory, and storage
  • Organizes files and folders
  • Runs multiple programs simultaneously
  • Controls input/output devices
  • Provides user interface (GUI)

Examples: Windows, macOS, Linux, Android

9. Quick Summary / Revision Notes

  • Computer = Input + Processing + Output + Storage
  • Memory Types: RAM, ROM, Cache, Secondary Storage
  • CPU/Microprocessor = Brain of computer
  • Data → Information: Raw data processed into meaningful information
  • Software: System (OS, Utilities) + Application (Word, Excel, etc.)
  • Operating System: Manages hardware, files, devices, processes, GUI

Tip for Exam: Remember the flow: Input → Memory → CPU → Processing → Output → Storage → Information. Understand data vs information, types of memory, software, OS, and generations.
