1. Introduction to Computer

A computer is an electronic device that can accept input, process it, store data, and produce output.

  • Accept input via devices (keyboard, mouse, scanner)
  • Process using CPU and instructions
  • Store temporarily or permanently
  • Produce output via monitor, printer, speakers
Example: Input marks of 5 subjects → Process: Total & Average → Output: Total=450, Average=90 → Storage: Saved for future

Characteristics of Computer:

  • Speed: Millions or billions of operations per second
  • Accuracy: Correct output if input/program correct
  • Automation: Works automatically as per instructions
  • Versatility: Multiple tasks like calculations, gaming, graphics
  • Storage & Retrieval: Store huge data & retrieve instantly
  • Connectivity: Can connect to networks (LAN, Internet)
  • Diligence: Works tirelessly
  • Multitasking: Run multiple programs simultaneously

Tip: Remember Input → Processing → Output → Storage for all questions!

3. Computer Memory

Memory is used to store data and instructions.

Primary Memory:

  • RAM: Volatile, stores running programs, example: Open Word doc
  • ROM: Non-volatile, stores boot instructions (BIOS)

Cache Memory:

  • Fast memory between CPU and RAM
  • Speeds up frequently used instructions

Secondary Memory:

  • Permanent storage: HDD, SSD, Pen Drive, CD/DVD
Memory Hierarchy (Speed ↓, Size ↑, Cost ↓)
Cache → RAM → Secondary Storage
Extra: Cache is small but extremely fast, often in KBs or MBs.

4. Data Transfer Between CPU & Memory

CPU communicates with memory using Buses:

  • Data Bus: Transfers actual data
  • Address Bus: Specifies memory location
  • Control Bus: Sends control signals (Read/Write)
Input → Memory ↔ CPU (ALU + CU) → Output → Storage
Tip: Learn the difference between Data, Address & Control bus.

5. Microprocessor

A microprocessor is a single-chip CPU that performs all operations.

  • ALU: Arithmetic & logic operations
  • CU: Directs flow of instructions & data
  • I/O Interface: Communicates with devices
Extra: Microprocessors are used in PCs, Laptops, Mobiles, Calculators, ATMs.

6. Data & Information

  • Data: Raw facts, letters, numbers, no meaning (e.g., 10, 20, A, B)
  • Information: Processed data, meaningful (e.g., Total Marks = 150)
Data → Processing → Information → Knowledge → Decision Making
Tip: Exam questions may ask differences between data & information.

7. Software

  • System Software: Helps computer operate
    • OS: Manages CPU, memory, storage, devices (Windows, Linux)
    • Utility Programs: Antivirus, Backup
  • Application Software: Helps user perform tasks (MS Word, Excel, PowerPoint)
HardwareSoftware
Physical componentsPrograms/instructions

8. Operating System (OS)

  • Manages hardware, memory, and storage
  • Organizes files & folders
  • Runs multiple programs simultaneously
  • Controls input/output devices
  • Provides GUI or CLI for users
Extra Tip: GUI = Icons, Menus, Buttons; CLI = Text commands

Examples: Windows, macOS, Linux, Android

9. Quick Summary / Revision Notes

  • Computer = Input + Processing + Output + Storage
  • Memory: RAM, ROM, Cache, Secondary Storage
  • CPU/Microprocessor: Brain → ALU + CU + I/O
  • Data → Information → Knowledge → Decision
  • Software: System + Application
  • OS Functions: Resource, File, Process, Device Management + User Interface
  • Generations: Vacuum Tubes → Transistors → IC → Microprocessors → AI/NLP
Tips: Draw diagrams, memorize generations table, remember memory hierarchy, learn difference between Data & Information, Hardware & Software.
